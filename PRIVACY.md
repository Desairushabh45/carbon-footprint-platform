# Privacy Policy for EcoTrack

## Data Collection
EcoTrack collects ZERO personal data.
All calculations happen in your browser.

## What We Store
- Anonymous carbon calculations in Firestore
- No names, emails, or personal info
- No cookies or tracking pixels

## API Keys
Your Gemini API key is:
- Never stored permanently
- Never sent to our servers  
- Destroyed after single use
- Session-only in memory

## Analytics
We use Google Analytics to track:
- Page views (anonymous)
- Feature usage (anonymous)
- No personal data collected
